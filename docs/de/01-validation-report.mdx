## Korrekturstrategien

| Strategie | Beschreibung |
|----------|-------------|
| **Gelockert** | Einschränkung war zu streng – Die API akzeptiert Werte, die von der Spezifikation abgelehnt wurden |
| **Verschärft** | Einschränkung war zu locker – Die API lehnt Werte ab, die von der Spezifikation akzeptiert wurden |
| **Hinzugefügt** | Fehlende Einschränkung wurde durch API-Tests entdeckt |
| **Entfernt** | Zusätzliche Einschränkung, die von der API ignoriert wird |
| **Aktualisiert** | Schema wurde an die API-Antwortstruktur angepasst |
