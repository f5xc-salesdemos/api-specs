# การแก้ไข API Spec ของ F5 XC ที่ถูกนำไปใช้

ตรวจสอบความถูกต้องและปรับปรุงข้อกำหนด OpenAPI ของ F5 Distributed Cloud

โปรเจกต์นี้ตรวจสอบความถูกต้องของข้อกำหนด OpenAPI ของ F5 XC เทียบกับ API จริง ระบุความแตกต่าง และนำการแก้ไขไปใช้โดยอัตโนมัติเพื่อสร้างไฟล์ spec ที่ถูกต้อง

*อัปเดตล่าสุด: 2026-06-07 08:58 UTC*

## สรุป

- **ไฟล์ที่แก้ไข**: 0
- **ไฟล์ที่ไม่ได้แก้ไข**: 0
- **ความแตกต่างทั้งหมด**: 0

:::tip[ไม่จำเป็นต้องแก้ไข]
spec ทั้งหมดตรงกับพฤติกรรมของ API จริง ไม่มีการแก้ไขใดถูกนำไปใช้
:::

## กลยุทธ์การแก้ไข

| กลยุทธ์ | คำอธิบาย |
|----------|-------------|
| **Relaxed** | ข้อจำกัดเข้มงวดเกินไป - API ยอมรับค่าที่ spec ปฏิเสธ |
| **Tightened** | ข้อจำกัดหลวมเกินไป - API ปฏิเสธค่าที่ spec ยอมรับ |
| **Added** | พบข้อจำกัดที่ขาดหายไปจากการทดสอบ API |
| **Removed** | ข้อจำกัดส่วนเกินที่ API ไม่สนใจ |
| **Updated** | อัปเดต schema ให้ตรงกับโครงสร้างการตอบกลับของ API |
